# SalesPulse dashboard

A polished, dependency-free browser dashboard built on the outputs of the restaurant-sales ML pipeline.

## Open it

Open `dashboard/index.html` in a modern browser. No package installation is required.

The dashboard incorporates the current generated model metrics, store and product forecasts, category trends, and customer segmentation. The Sales AI panel is a local, data-grounded assistant for exploring those results; it does not call an external AI API.

For a full AI service in production, connect the assistant endpoint to an approved LLM provider and pass only governed, warehouse-derived aggregate data.
